DRMQ is a distributed message broker built for guaranteed delivery, strict ordering, and high availability. It uses the Raft consensus algorithm to keep your cluster coordinated and your messages safe — even when nodes fail. Whether you're running a single node for development or a three-node production cluster, DRMQ gives you scalable consumer groups, dead-letter queues, and native clients in Java, Python, and TypeScript.

## What You Can Build

DRMQ is designed for use cases where message reliability and ordering matter:

- **Event-driven microservices** — decouple producers and consumers with at-least-once delivery guarantees
- **Work queues** — distribute jobs across multiple workers using consumer groups, no partitions required
- **Audit logs** — persist ordered message streams with configurable retention
- **Replay pipelines** — rewind and replay messages from any offset with single-consumer mode

## Get Started

<Steps>
  <Step title="Build DRMQ">
    Clone the repository and build all modules with Maven.
    ```bash
    git clone https://github.com/samuel025/DRMQ.git
    cd DRMQ
    mvn clean install
    ```
  </Step>
  <Step title="Start the broker">
    Launch a single-node broker on port 9092.
    ```bash
    ./mvnw -pl drmq-broker exec:java -Dexec.args="--port 9092 --data-dir ./data-1"
    ```
  </Step>
  <Step title="Send a message">
    Use the Java client to produce your first message.
    ```java
    try (DRMQProducer producer = new DRMQProducer("localhost:9092")) {
        producer.connect();
        DRMQProducer.SendResult result = producer.send("my-topic", "Hello, DRMQ!");
        System.out.println("Sent at offset " + result.getOffset());
    }
    ```
  </Step>
  <Step title="Consume messages">
    Subscribe and poll for messages with the consumer client.
    ```java
    try (DRMQConsumer consumer = new DRMQConsumer("localhost:9092", "my-group")) {
        consumer.setAutoCommit(true);
        consumer.connect();
        consumer.subscribe("my-topic");
        List<DRMQConsumer.ConsumedMessage> msgs = consumer.poll(100, 1000);
        msgs.forEach(m -> System.out.println(m.payloadAsString()));
    }
    ```
  </Step>
</Steps>
